# MBTI职业性格测试系统

## 项目简介
基于迈尔斯-布里格斯类型指标（MBTI）的完整版93题职业性格测试系统。

## 功能特性
- **用户端**：93道题目测试，答题完成后仅显示完成提示
- **管理后台**：统一登录入口，查看所有人测试数据和个人详细报告
- **权限隔离**：用户无法查看结果，仅管理员可查看分析报告

## 部署方式
本项目已配置GitHub Pages自动部署。
